# Bộ tài liệu hướng dẫn dự án Star-cubing

Thư mục này chứa các tài liệu vận hành và quy định chuẩn cho dự án khai phá dữ liệu Iceberg Cube bằng thuật toán Star-cubing.

## Danh mục tài liệu

| Tài liệu | Mục đích | Mức độ ưu tiên đọc |
|---|---|---|
| 01_Huong_dan_thiet_lap.md | Thiết lập môi trường, cấu hình và chạy dự án lần đầu | Cao |
| 02_Quy_dinh_du_lieu.md | Quy định chuẩn dữ liệu đầu vào/đầu ra, chất lượng dữ liệu | Cao |
| 03_Quy_dinh_code.md | Quy ước code Python/SQL, cấu trúc thư mục, checklist review | Cao |
| 04_Quy_trinh_star_cubing.md | Quy trình thực thi Star-cubing từ dữ liệu đến cube kết quả | Trung bình |

## Nguyên tắc sử dụng tài liệu

1. Khi có thay đổi schema hoặc logic tính toán, phải cập nhật tài liệu tương ứng ngay.
2. Mọi thành viên cần tuân thủ quy định dữ liệu trước khi commit code.
3. Ưu tiên tái sử dụng cấu trúc file và template trong các tài liệu này để đảm bảo nhất quán.
